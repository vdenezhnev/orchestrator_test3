# Инженерный калькулятор — Анализ требований и прототип

## Описание проекта

Веб-приложение инженерного калькулятора с базовыми и научными математическими функциями.

## Структура репозитория

```
/
├── docs/
│   ├── requirements.md          # Функциональные и нефункциональные требования
│   ├── engineering-functions.md # Детальное описание инженерных функций
│   └── wireframes/
│       ├── wireframe-description.md  # ASCII-wireframe и описание компонентов
│       └── style-guide.md            # Руководство по стилю (цвета, шрифты, отступы)
├── prototype/
│   ├── index.html               # HTML-структура калькулятора
│   ├── styles.css               # Стили (светлая/темная тема)
│   └── script.js                # Базовая логика калькулятора
└── README.md                    # Этот файл
```

## Функциональность прототипа

### Базовые операции
- Сложение, вычитание, умножение, деление
- Процент и изменение знака
- Десятичные числа

### Инженерные функции
- **Тригонометрия**: sin, cos, tan, asin, acos, atan
- **Логарифмы**: log, ln, 10^x, e^x
- **Степени**: x², x³, √x, ∛x
- **Специальные**: |x|, n!, 1/x, π, e

### Дополнительные возможности
- Переключение DEG/RAD
- Функции памяти (MC, MR, M+, M-, MS)
- История вычислений
- Темная/светлая тема
- Поддержка клавиатуры

## Запуск прототипа

Откройте `prototype/index.html` в браузере. Внешние зависимости не требуются.

```bash
# Или используйте локальный сервер
cd prototype
python3 -m http.server 8080
# Откройте http://localhost:8080
```
