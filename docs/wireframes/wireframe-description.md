## Компоненты интерфейса

### 1. Header
- Переключатель темы (светлая/темная)
- Кнопка истории вычислений

### 2. Display (Экран)
- Верхняя строка: текущее выражение
- Нижняя строка: результат (крупный шрифт)
- Индикатор активной памяти (M)
- Индикатор режима (DEG/RAD)

### 3. Mode Selector
- DEG/RAD — переключение угловых единиц
- 2nd — доступ к дополнительным функциям

### 4. Memory Buttons (Кнопки памяти)
- Горизонтальная панель с 5 кнопками
- Визуальный индикатор при наличии значения в памяти

### 5. Scientific Functions (Научные функции)
- Сетка 6×4 кнопок
- Цветовое кодирование по категориям
- Hover-эффекты для интерактивности

### 6. Basic Keypad (Базовая клавиатура)
- Классическая раскладка калькулятора
- Кнопка "0" занимает двойную ширину
- Кнопка "=" выделена цветом

## Адаптивность

### Desktop (> 768px)
- Полная раскладка с научными функциями
- Боковая панель истории (опционально)

### Tablet (768px - 480px)
- Компактная раскладка
- Научные функции в выпадающей панели

### Mobile (< 480px)
- Базовый калькулятор по умолчанию
- Свайп для доступа к научным функциям
- Увеличенные кнопки для удобства касания

## Состояния кнопок

1. **Default** — обычное состояние
2. **Hover** — наведение курсора
3. **Active** — нажатие
4. **Disabled** — недоступна (например, MR при пустой памяти)

## Анимации

- Нажатие кнопки: scale(0.95), 100ms
- Смена темы: transition 300ms
- Появление результата: fade-in 200ms
- Открытие панели истории: slide-in 250ms
