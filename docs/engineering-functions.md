# Набор инженерных функций калькулятора

## Категоризация функций

### Категория 1: Базовая арифметика (Приоритет: Критический)

| Функция | Символ | Описание | Пример |
|---------|--------|----------|--------|
| Сложение | + | Сумма двух чисел | 5 + 3 = 8 |
| Вычитание | - | Разность двух чисел | 5 - 3 = 2 |
| Умножение | × | Произведение двух чисел | 5 × 3 = 15 |
| Деление | ÷ | Частное двух чисел | 15 ÷ 3 = 5 |
| Процент | % | Процент от числа | 50% × 200 = 100 |

### Категория 2: Тригонометрия (Приоритет: Высокий)

| Функция | Описание | Диапазон значений |
|---------|----------|-------------------|
| sin(x) | Синус угла | [-1, 1] |
| cos(x) | Косинус угла | [-1, 1] |
| tan(x) | Тангенс угла | (-∞, +∞) |
| asin(x) | Арксинус | [-π/2, π/2] или [-90°, 90°] |
| acos(x) | Арккосинус | [0, π] или [0°, 180°] |
| atan(x) | Арктангенс | (-π/2, π/2) или (-90°, 90°) |

**Режимы:**
- DEG — градусы (по умолчанию)
- RAD — радианы

### Категория 3: Логарифмы и экспоненты (Приоритет: Высокий)

| Функция | Описание | Формула |
|---------|----------|---------|
| log(x) | Десятичный логарифм | log₁₀(x) |
| ln(x) | Натуральный логарифм | logₑ(x) |
| e^x | Экспонента | 2.71828...^x |
| 10^x | Степень десяти | 10^x |

### Категория 4: Степени и корни (Приоритет: Высокий)

| Функция | Символ | Описание |
|---------|--------|----------|
| Квадрат | x² | x × x |
| Куб | x³ | x × x × x |
| Степень | xⁿ | x в степени n |
| Кв. корень | √x | Квадратный корень |
| Куб. корень | ³√x | Кубический корень |
| Корень n-ой степени | ⁿ√x | Корень степени n |

### Категория 5: Специальные функции (Приоритет: Средний)

| Функция | Символ | Описание |
|---------|--------|----------|
| Модуль | \|x\| | Абсолютное значение |
| Факториал | n! | n × (n-1) × ... × 1 |
| Число Пи | π | 3.14159265358979... |
| Число Эйлера | e | 2.71828182845904... |
| Обратное число | 1/x | Единица, делённая на x |
| Научная нотация | EXP | a × 10^n |

### Категория 6: Память (Приоритет: Средний)

| Кнопка | Действие |
|--------|----------|
| MC | Очистка памяти (Memory Clear) |
| MR | Вызов значения из памяти (Memory Recall) |
| M+ | Добавление к значению в памяти |
| M- | Вычитание из значения в памяти |
| MS | Сохранение в память (Memory Store) |

### Категория 7: Управление (Приоритет: Критический)

| Кнопка | Действие |
|--------|----------|
| C | Очистка текущего ввода |
| AC | Полная очистка |
| ⌫ | Удаление последнего символа |
| = | Вычисление результата |
| +/- | Смена знака числа |

## Приоритет операций

1. Скобки ()
2. Функции (sin, cos, log, etc.)
3. Степень (^)
4. Умножение и деление (×, ÷)
5. Сложение и вычитание (+, -)

## Обработка ошибок

| Ситуация | Сообщение |
|----------|-----------|
| Деление на ноль | "Error: Division by zero" |
| √ отрицательного числа | "Error: Invalid input" |
| log(0) или log(отр.) | "Error: Invalid input" |
| Переполнение | "Error: Overflow" |
| Некорректный ввод | "Error: Syntax error" |
