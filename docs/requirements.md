# Требования к инженерному калькулятору

## 1. Общее описание

Инженерный калькулятор — веб-приложение для выполнения базовых и научных математических вычислений.

## 2. Функциональные требования

### 2.1 Базовые операции
- Сложение (+)
- Вычитание (-)
- Умножение (×)
- Деление (÷)
- Процент (%)
- Изменение знака (+/-)

### 2.2 Инженерные функции

#### Тригонометрия
- sin, cos, tan — тригонометрические функции
- asin, acos, atan — обратные тригонометрические функции
- Переключение режима: градусы (DEG) / радианы (RAD)

#### Логарифмы и экспоненты
- log — десятичный логарифм
- ln — натуральный логарифм
- e^x — экспонента
- 10^x — степень десяти

#### Степени и корни
- x² — квадрат числа
- x³ — куб числа
- xⁿ — произвольная степень
- √x — квадратный корень
- ³√x — кубический корень
- ⁿ√x — корень n-ой степени

#### Специальные функции
- |x| — модуль числа
- n! — факториал
- π — число пи
- e — число Эйлера
- 1/x — обратное число
- EXP — экспоненциальная запись (научная нотация)

### 2.3 Работа с памятью
- MC — очистка памяти
- MR — вызов из памяти
- M+ — добавление к памяти
- M- — вычитание из памяти
- MS — сохранение в память

### 2.4 Управление вводом
- C — очистка текущего ввода
- AC — полная очистка (All Clear)
- ⌫ — удаление последнего символа (Backspace)
- = — вычисление результата

## 3. Нефункциональные требования

### 3.1 Интерфейс
- Адаптивный дизайн (десктоп и мобильные устройства)
- Темная и светлая темы
- Поддержка клавиатурного ввода
- История вычислений

### 3.2 Производительность
- Мгновенный отклик на ввод (< 100ms)
- Корректная работа с большими числами
- Обработка ошибок (деление на ноль, переполнение)

### 3.3 Точность вычислений
- Точность до 15 значащих цифр
- Корректное округление
- Отображение в научной нотации для очень больших/малых чисел

## 4. Целевая аудитория

- Студенты технических специальностей
- Инженеры и ученые
- Разработчики программного обеспечения

## 5. Технический стек

- HTML5, CSS3, JavaScript (Vanilla JS)
- Без внешних зависимостей для базового функционала
- Возможность расширения с использованием фреймворков
